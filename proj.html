Deinflection Deinflection is a C#-based library that allows deinflection of inflected Japanese verbs and adjective as well as the types of inflection being used.

The library normally outputs the dictionary form of the verb/adjective and the series of inflections being used on it.

This library utilizes a Inflection Method - to - Character Set approach to be able to extract the corresponding character set for each type of inflection made on the verb.

Phase 1: Simple Inflection.

On this project phase, it can only return the dictionary form of the verb and the single inflection type used.
eg. 使います -> {'使う', 'present polite'}

Phase 2: Series Inflection.
On this phase, it can return the dictionary form and the series of the inflections used on the verb.
eg 使わないで -> {'使う', ['conjunctive', 'negative']}
Deconjugator A C#-based library similar to Deinflection but for Korean verbs.

The main difference with Deinflection is that character manipulation is based on mathematical computation of characters instead of a Conjugation-To-Character Set Approach.
